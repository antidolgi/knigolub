    💡 ИДЕИ ДЛЯ ОБСУЖДЕНИЯ

    🧮 КАЛЬКУЛЯТОР ДЛЯ ТК

    Зачем нужен:

    • Быстро рассчитывать вес и объем партии книг
    • Давать точные данные в транспортные компании
    • Считать себестоимость доставки на 1 книгу

    Функционал:

    • Ввод: вид книги (твердый/мягкий переплет), количество
    • База: вес, размеры по категориям
    • Расчет: общий вес (кг), объем (м³), количество мест
    • Вывод: PDF с данными для ТК
    • Второй калькулятор: цена доставки + цена книг = себестоимость

    Реализация: Попробую сделать сам (Excel/Google Sheets), если сложно — привлечь программиста
